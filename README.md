# ⏰ Digital Clock

## 📌 Overview
This is a simple real-time digital clock built using HTML, CSS, and JavaScript.  
The clock dynamically updates every second using JavaScript's Date object.


## 🚀 Features
- Displays current system time
- Updates automatically every second
- Clean and minimal user interface
- Lightweight and responsive design


## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript


## 🔧 How It Works
The clock uses:
- `setInterval()` to update time every second
- `Date()` object to fetch the current time
- DOM manipulation to display the updated time
